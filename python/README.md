# Adaptive Echo (Python Implementation)

## Installation

Install `uv`: https://docs.astral.sh/uv/

Install dependencies:

```bash
uv sync
```

## Usage

Generate some random sounds from the synthesizer:

```bash
uv run python src/adaptive_echo/synth.py
```

## Testing

```bash
./test.sh

```
